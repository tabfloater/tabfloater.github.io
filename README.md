# TabFloater.io

Source code for https://www.tabfloater.io. This site uses [Jekyll](https://jekyllrb.com/) with GitHub Pages.

## Developing

 * [Set up](https://jekyllrb.com/docs/installation/) Jekyll on your machine
 * Execute `serve.sh`
